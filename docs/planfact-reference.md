# Референс интерфейса ПланФакт (снято с app/demo.planfact.io)

Карта меню, структуры страниц и дизайн-токены — основа для доведения клона до 1:1.
Снято headless-браузером (Playwright) с авторизованного кабинета и демо-бизнеса.

## Навигация (сайдбар, иконки + подписи)
- **Показатели** `/` — дашборд
- **Операции** `/transactions`
- **Сделки**: Продажи `/deals/selling`, Закупки `/deals/purchase`, Выставленные счета `/invoice-documents`
- **План**: Платёжный календарь `/budget/planning`, Бюджет доходов и расходов (БДР) `/budget/Bdr`, Бюджет движения денег (БДДС) `/budget/Bdds`
- **Проекты** `/projects`
- **Отчёты**: Движение денег (ДДС) `/reports/cashflow`, Прибыли и убытки (ОПУ) `/reports/pnl`, Баланс `/balance`
- **Справочники**: Контрагенты `/contractors`, Учётные статьи `/transaction-categories`, Мои счета `/accounts`, Мои юрлица `/legal-entities`, Товары `/products`, Услуги `/products/services`
- **Настройки** `/manage/general-settings`

## Дизайн-токены
- Логотип «ПФ» — белый в бирюзовом квадрате. Акцент — **бирюзовый/teal** (≈ `#16b1bf`/`#17a2b8`),
  активные тоглы — рамка teal, ссылки teal.
- Сайдбар — тёмный сине-серый (≈ `#2b3b4e`), пункт = иконка сверху + подпись снизу, активный светлее.
- Верхняя панель (тёмная): слева «Помощь и обучение», по центру «**На счетах N ₽** / Разрыв c дд.мм по дд.мм»
  + мини-спарклайн остатка, справа e-mail + «Подписка активна», колокол, кнопка «Подписка».
- Контент — белый; шапки таблиц — мелкий серый текст; разделы — жирные строки; первый столбец «липкий»;
  суммы справа, цветные (зелёный +, красный −). Числа: пробел-разделитель тысяч, «−» для минуса.
- На списках/отчётах слева — панель **Фильтры** (светлая, сворачивается стрелкой).

## Дашборд (Показатели)
Фильтры: Период (диапазон), Отображение (По месяцам), Юрлица и счета, Проекты, Сделки.
Блоки:
1. **Прибыль, ₽** — тогл «Метод начисления / Кассовый метод». Слева: Доходы, Расходы, Чистая прибыль,
   Рентабельность %, Дивиденды (большие числа + синее «… по плану»). Справа: бар-чарт (Доходы — голубые,
   Расходы — оранжевые, будущее — пунктир), линии Чистая прибыль (зелёная), Дивиденды (фиолетовая), слайдер периода.
2. **Денежный поток, ₽** — вкладки Общий/Операционный/Инвестиционный/Финансовый. Поступления, Выплаты, Разница + график.
3. **Остатки на счетах, ₽** — линейный график по счетам + общий остаток.
4. **Структура платежей** — вкладки Доходы и расходы / Поступления и выплаты; пончик + бар по статьям.
5. **Самые доходные клиенты, ₽** — ABC-анализ (накопительный итог, >80% / <20%).
6. **Прибыльность проектов** — бар прибыль + линия рентабельности.

## Операции `/transactions`
- Левые фильтры: **Тип операции** (Поступление, Выплата, Перемещение, Начисление, Отгрузка, Поставка),
  **Дата оплаты** (Подтверждена/Не подтверждена + период), **Дата начисления** (то же),
  Параметры: Юрлица и счета, Контрагенты, Статьи учёта, Проекты, Сделки.
- Таблица: ☑ | Дата | Счёт | Тип (иконка-стрелка, цветная) | Контрагент | Статья | Проект | Сделка | Сумма.
- Группировка по датам («Сегодня нет операций», «Вчера и ранее»).
- Итоговая строка снизу: «N операций | поступления: ₽ | выплаты: ₽ | перемещения: ₽ | начисления | Итого: ₽».

## Отчёт ДДС `/reports/cashflow`
Шапка: «Отчет о движении денежных средств», бейдж валюты, «Способ построения: По периодам», «Отображение: По месяцам».
Столбцы: «По статьям учёта ▾» | **Тренд** (спарклайн) | месяцы… | Итого. Строки (разворачиваемые):
- **Операционный поток** (Поступления / Выплаты)
- **Инвестиционный поток** (Поступления / Выплаты)
- **Финансовый поток** (Поступления / Выплаты)
- **Перемещения** (Списания / Зачисления)
- **Общий денежный поток** (итог)
- **Остатки на конец периода** (по юрлицам)
→ Статья имеет признак **вида деятельности** (operating/investing/financing).

## Отчёт ОПиУ `/reports/pnl`
Столбцы как в ДДС (со «Трендом»). Строки:
Выручка → Прямые расходы → **Операционная прибыль** / **Операционная рентабельность** →
Косвенные расходы → Прочие доходы → Прочие расходы → Амортизация →
Проценты по кредитам и займам → Налоги на прибыль → **Чистая прибыль**.
Тогл метода: начисления / кассовый.

## Учётные статьи `/transaction-categories`
Вкладки: **Доходы | Расходы | Активы | Обязательства**. Внизу: Чистая прибыль, Итого активы,
Итого обязательства, Итого капитал (нераспределённая прибыль). → статьи 4 типов (для ОПиУ, ДДС и Баланса).

## Справочники — колонки
- **Контрагенты**: Контрагент | Тип | Группа | ИНН | Операций | Дебиторка ₽ | Кредиторка ₽ | Поступления ₽ | Выплаты ₽ | Разница ₽.
- **Счета**: Название | Группа | Начальный остаток | Текущий остаток | Тип | Реквизиты.
- **Юрлица**: Краткое название | Полное название | ИНН | КПП.
- **Товары**: Наименование | Артикул | Цена за ед. | Единица | НДС | Цена с НДС | Комментарий.
- **Услуги**: как товары без «Артикул».
- **Проекты**: Название | Группа | Начало/Конец | Статус | Доходы ₽ | Расходы ₽ | Прибыль ₽ | Рентабельность.

## Сделки `/deals/selling` (и `/purchase`)
Колонки: Дата | Название | Клиент | Статус | Сумма сделки ₽ | Поступило | Отгружено | Прибыль ₽.
Воронка статусов. Закупки — аналогично («Поставщик» вместо «Клиент»).

## План
- **Платёжный календарь** `/budget/planning`: сетка Доходы/Расходы по месяцам, план vs факт.
- **БДР/БДДС** `/budget/Bdr|Bdds`: список бюджетов (Название | Валюта | Юрлицо | Проект | Период | Дата изменения | Кто изменил),
  внутри — сетка планирования по статьям × месяцам с план-фактом.
